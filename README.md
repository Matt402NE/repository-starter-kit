## Introduction
Welcome to the **Repository Starter Kit**  This is an [Obsidian](https://obsidian.md/) vault focused on AI assisted development. This project includes customizable instructions and prompts to help development teams identify and mitigate security risks effectively.
  
## How To Use

1. Fork the starter kit
   Create a new repository using the `starter kit` as the template.
2. Clone the new repository
3. Rename the `github` folder to `.github` (add period).
4. Rename the `vscode` folder to `.vscode` (add period). 
5. Delete the `.obsidian` folder.  It is a folder used by Obsidian.
6. **Update the instructions** 
   - Review the `.github/instructions/` folder for coding rules and best practices. Update them and make them your own.
